🚨 The Problem

Modern SMEs and smart campuses face increasing cyber threats such as DDoS attacks, but:

-> Detection systems are reactive, not proactive

-> Security logs lack real-time visibility

-> Alerts are not risk-prioritized

-> Cloud-only models increase latency and privacy concerns

Organizations need intelligent, low-latency, edge-based security.

💡 Our Solution

SentinelEdge AI is a real-time AI-powered intrusion detection system that:

-> Detects cyber threats using Machine Learning

-> Calculates dynamic risk scores

-> Classifies severity levels instantly

-> Visualizes live attack telemetry

-> Benchmarks Edge vs Cloud latency

All in one unified security intelligence platform.

🧠 What Makes It Innovative?
🔹 Dual-Model AI Engine

We combine:

-> 🌲 Random Forest (Supervised Attack Classification)

-> 🧊 Isolation Forest (Unsupervised Anomaly Detection)

This hybrid approach improves reliability over single-model systems.

🔹 Dynamic Risk Scoring Engine
Risk Score = 100 × (0.55 × anomaly_score + 0.45 × confidence)

This ensures:

-> High anomaly + high confidence = Critical alert

-> Intelligent alert prioritization

-> Reduced false alarm fatigue

🔹 Edge vs Cloud Benchmarking

Our system measures:

⚡ Edge inference latency

☁ Cloud processing latency

📊 Real-time performance comparison

Demonstrates the efficiency of edge computing for cybersecurity.

Train Models
  python backend/train.py
Start Backend
  uvicorn backend.app:app --reload
Run Streamer
  python streamer/stream.py
Launch Dashboard
  streamlit run ui/dashboard.py
